# Privacy Policy for AIBlocker

**Last updated: June 20, 2026**

AIBlocker is a privacy-first browser extension designed to block chatbots, hide AI Overview sections on Google search results, and detect AI-modified videos on YouTube.

### 1. No Data Collection
AIBlocker does not collect, store, or transmit any user data, personal information, browsing history, search queries, or visited URLs. All processing and blocking actions are performed locally on your device.

### 2. Local Processing
All extension features, including the detection of chatbot scripts, search result cleanups, and YouTube video filtering, run entirely on the client side inside your browser. No data is sent to external servers.

### 3. Remote Configuration
The extension periodically downloads a static JSON configuration file from a public GitHub repository containing updated blocking rules. No user-identifiable information, cookies, or tracking headers are sent during this request.

### 4. Contact
If you have any questions about this Privacy Policy, you can open an issue at:
